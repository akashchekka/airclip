How to Embed

Standard mode uses <video>, transparent mode uses <img>. No JavaScript needed for either.

<!-- Standard mode: WebM with <video> -->
<video autoplay muted loop playsinline
       style="width:100%; border-radius:10px">
    <source src="animation.webm" type="video/webm">
</video>

<!-- Transparent mode: WebP with <img> -->
<img src="animation.webp" style="width:100%">
# Standard mode
python -m airclip input.mp4

# Transparent mode (source must have alpha channel)
python -m airclip animation.mov --transparent --crf 28
