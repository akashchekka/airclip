Inline with Text

Videos small enough to float inside paragraphs — like images, but animated.

A qubit can exist in a superposition of |0⟩ and |1⟩ simultaneously. Every single-qubit state maps to a point on the surface of the Bloch sphere. The north pole represents |0⟩, the south pole |1⟩, and the equator holds all equal superpositions like |+⟩ and |−⟩.

Quantum gates are simply rotations on this sphere. An X gate flips north↔south. A Hadamard rotates from the pole to the equator. The animation on the right is only 123 KB — small enough to embed anywhere without impacting page load.

This is the power of NanoClip: videos so small they behave like images. No lazy loading needed. No CDN required. Just drop them in and they blend.

How to Embed

One HTML tag. Background matches automatically. No JavaScript needed.

<!-- Seamless: no controls, autoplay on scroll -->
<video autoplay muted loop playsinline
       style="width:100%; background:#1a1a2e; border-radius:10px">
    <source src="animation.webm" type="video/webm">
</video>

<!-- With controls on hover (CSS) -->
<style>
  video::-webkit-media-controls { opacity: 0; transition: opacity 0.3s; }
  video:hover::-webkit-media-controls { opacity: 1; }
</style>
# Convert any video
python -m nanoclip input.mp4

# Custom settings
python -m nanoclip input.mp4 --fps 15 --crf 40 --height 480
